# 收割(Harvest)

### Harvest

| 设置参数                                                                     | Ethereum      | Polygon       |
| ------------------------------------------------------------------------ | ------------- | ------------- |
| 定时任务触发周期                                                                 | 每日 22:00（UTC） | 每日 22:00（UTC） |
| 触发策略`harvest`最大时间间隔 （若当前`harvest`与上次`harvest`或`lend`间隔大于此值，必须做`harvest`） | 3天            | 1天            |
| 触发策略`harvest`的收益成本比例系数X （`harvest`收益 >= 成本\*X，则可做`harvest`）              | 5             | 5             |
